Dieses Addon ist zuständig für Anpassungen an folgenden Reports:
  1. Invoice (Inherit-ID: account.report_invoice_document)
    • VAT/USt. des Kunden im Adress-Fenster ausgeblendet
    • Customer Code wird nur bei Kunden mit der Sprache en_US ausgegeben
    • Quelle (Verweiße von Angebote) ausblenden
    • Spalte Steuern ausgeblendet
    • Verkäufer eingebunden
  2. Sale/Order (Inherit-ID: sale.report_saleorder_document)
    • VAT/USt. des Kunden im Adress-Fenster ausgeblendet
    • DateTime-Feld "Bestelldatum" in Date umgewandelt
    • Spalte Steuern ausgeblendet
    • Verkäufer eingebunden
    • Steuern-Summe im Block "Total" ausgeblendet
    • Spalten-Breiten des Blocks "Total" vergrößert
