Voraussetzung für dieses Reports-Theme ist die Verwendung der Vorlage DIN 5008 unter:
  1. Einstellungen (Entwicklermodus aktivieren)
  2. Allgemeine Einstellungen
  3. Geschäftsunterlagen => Dokumenten-Layout
  4. Layout: external_layout_din5008 (l10n_de.external_layout_din5008)

Hinweis: Das Reports-Logo wird nicht von der Webseite herangezogen, sondern liegt im Addons-Ordner reports_theme_gab/static/src/img/reports_logo.png
des Weiteren wird die INN und KPP für russische Rechnungen, Angebote und Aufträge eingebunden.

Weitere Informationen zu Maßeinheiten der Vorlage nach DIN 5008 (Form B) finden Sie auf Wikipedia.
